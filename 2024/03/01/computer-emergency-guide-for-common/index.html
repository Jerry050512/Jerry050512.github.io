Windows 电脑应急指南 - 通用篇

发表于2024-03-01|更新于2024-05-12|Computer-Emergency-Guide
|阅读量:

写在前面

无论怎么样，最重要的是要定期备份数据，尤其是尽量不要把重要数据放在系统盘（一般是C盘），因为即使重装系统也只需格式化C盘。

另外应该定期创建系统还原点，以便不时之需。

常见解决方案

  1. 启动类修复工具可以使用系统自带的启动修复工具或一些第三方工具来修复启动项无法打开的问题。例如使用Windows安装U盘中的“启动修复”功能,它可以帮助扫描系统并修复启动故障。
  2. 检查事件日志在事件查看器中查看系统日志、应用程序日志和安全日志,看是否有关于错误的事件日志记录,这可以帮助定位问题原因。
  3. sfc 和 dism命令可以使用sfc /scannow命令扫描系统文件,然后用dism /online /cleanup-image /restorehealth命令修复损坏的系统文件。这可以修复缺失或损坏的系统文件问题。
  4. 修复注册表可以备份后编辑注册表,删除损坏的注册表键值。也可以使用一些注册表清理工具帮助修复注册表。
  5. 检查硬盘及驱动可以运行磁盘检查实用程序如chkdsk,检查磁盘错误。更新驱动程序版本或重装驱动也可能修复驱动故障导致的问题。
  6. 还原系统还原点如果问题是最近发生,可以系统还原到之前正常工作的还原点,撤销当前更改。

通过综合这些方法,可以解决启动菜单无法打开、程序无法运行的系统故障。

p.s. 无论你怎么操作过，都可以尝试重启一下，有时重启是可以解决绝大部分的问题的，但也要提防引起无法开机的问题。

进入安全模式

在有些时候如果你遇到无法开机/登录的问题，或者是正常系统下有什么问题，可以尝试进入安全模式。

如果在 Windows 10 及以上系统，按照以下步骤进入安全模式：

  • 对于已经登录到系统内的用户:
    • 在 电源 -> 重启 时, 按住 Shift 键, 等待重启, 随后按照 疑难解答 -> 高级选项 -> 启动设置 -> 重新启动 进入.
    • 或者, 在设置中找到高级启动进入.
  • 对于无法登录的用户:
    • 自 Windows 10 开始, 只能通过异常关机进入, 即在开机时出现徽标界面, 下方进度条转动时, 按住电源键强制关机, 重复3-4次, 直到界面变成执行 Windows 自动修复.

chkdsk

chkdsk是Windows系统中的磁盘检查和修复实用程序。

chkdsk全称为Check Disk,它可以用来检查磁盘上的文件系统完整性,并定位及修复磁盘错误。

chkdsk的主要功能包括:

  • 检测磁盘上的坏道,并将其映射到好道上。
  • 定位磁盘上的坏扇区,并尝试修复。
  • 检查文件系统的完整性,修复因意外关机、断电等导致的文件系统错误。
  • 扫描磁盘上的坏簇,尝试修复。
  • 修复目录结构错误,如文件夹循环引用等。
  • 修复安全描述符损坏导致的无法访问文件问题。
  • 恢复丢失的文件碎片。

chkdsk可以帮助我们解决由于磁盘错误导致的数据丢失或文件系统故障问题。在 Windows 中,我们可以在命令提示符使用chkdsk [磁盘代号] 参数的形式来运行它,也可以在Properties菜单中以图形方式启动chkdsk。定期使用chkdsk检查磁盘可以及时发现和修复问题,防止故障扩大。

用法

chkdsk的具体用法如下:

  1. 基本语法

    chkdsk [驱动器代号] [参数]

    例如:chkdsk C:

    这将检查C盘

  2. 常用参数

    /f - 修复磁盘上的错误

    /r - 定位坏扇区并恢复可读信息

    /x - 强制卸载卷,可用来检查使用中的磁盘

    /b - ClearDOS 位清空磁盘上DOS文件系统标志

    /scan - 扫描坏道或文件系统结构错误

  3. 修复文件系统错误

    chkdsk C: /f

    此命令将修复C盘文件系统错误

  4. 无提示检查并修复错误

    chkdsk C: /f /x

  5. 重写不可读取的扇区

    chkdsk C: /r

    此命令将恢复不可读的扇区

  6. 创建自检列表

    chkdsk C: /scan > scan.txt

    这样可以将结果存入文本文件

正确使用chkdsk参数可以实现对磁盘问题的检查定位和修复。但在运行前要注意做好数据备份,因为存在损坏数据的风险。

系统还原点

创建

创建系统还原点的方法如下:

  1. 通过控制面板

    • 进入控制面板 - 系统与安全 - 系统,点击左侧的“系统保护”
    • 在系统保护界面点击“创建”来手动创建一个还原点

  2. 通过命令提示符

    • 以管理员身份打开命令提示符

    • 使用命令:

      sysprep /createRestorePoint:"还原点描述"

    这样就可以创建一个带描述信息的还原点了

  3. 通过 Powershell

    • 打开Powershell,运行:

      Checkpoint-Computer -Description "还原点描述"

还原

  1. 选择一个要还原到的还原点,记录下还原点的序列号。

  1. 重启电脑后,还原点就会生效,系统会恢复到指定的还原点状态。

需要注意的是,还原系统还原点会撤销该还原点创建后所做的更改,可能会导致数据丢失,所以在还原前最好先备份重要数据。

命令行还原还原点是一个很方便的方法,当图形界面无法正常打开时,可以使用这种方式快速恢复系统。

重装系统

重装系统是修复系统最后也是最有效的方式，重装系统前要注意的是系统盘（通常是C盘）上的所有文件将被格式化，因此要做好备份。

基本步骤

重装Windows系统的主要步骤如下:

  1. 准备工作
    • 备份重要数据。将自己的数据、文档、图片等重要内容备份到其他储存设备中。
    • 准备安装U盘或DVD。使用Windows创建工具在U盘或刻录DVD来创建可启动的Windows安装介质。
    • 清理磁盘分区。使用磁盘管理清理系统盘分区,只保留一个分区用于安装系统。
  2. 进入BIOS,设置从U盘或DVD启动。
  3. 使用安装介质启动,进入Windows安装界面。
  4. 在安装类型界面选择“自定义安装”。
  5. 在磁盘分区界面删除所有旧分区,并创建新的系统分区。
  6. 按提示完成安装,等待系统安装完成。
  7. 安装完成后,进入Windows进行产品激活和相关驱动安装。
  8. 最后重新安装需要的程序,恢复数据文件。

需要注意的是,重装会清空原有的数据,所以必须提前做好备份。另外要准备好产品密钥进行激活。

详细说明

制作U盘启动盘

  1. 从官方网站下载Windows的系统镜像。

    • 访问下載 Windows 11 (microsoft.com)。
    • 选择 下载 Windows 11 磁盘映像(ISO) → Windows 11 (multi-edition ISO) → 简体中文
    • 等待下载，请注意文件应该是.iso的后缀名

    p.s. 也可以从msdn i tell you 等网站下载镜像，其可靠性自行判断。

  2. 使用镜像刷写软件将镜像输入U盘

    • 例如：Rufus - 轻松创建 USB 启动盘
    • 注意分区是GPT还是MBR

进入BIOS

BIOS 是 Basic Input/Output System，这里有一些电脑主板的基本选项。

修改BIOS启动项的主要步骤如下:

  1. 搜索进入BIOS的方式。
    • 重启电脑,在开机时按下BIOS快捷键(DEL、F1、F2等)进入BIOS设置菜单。
    • 通常可以使用 主板/电脑品牌 + 进入BIOS 来搜索
    • 例如HP的电脑在启动时按Esc键，选择启动项就可以从U盘启动
  2. 修改BIOS中的引导启动项顺序。
    • 在BIOS菜单中找到"Boot"或"Boot Order"选项。
    • 在Boot Order菜单中可以看到Boot Option Priorities选项,列出了启动设备的优先级。
    • 使用向上、向下箭头键调整启动设备的优先级顺序。将需要的启动设备移到最高优先级。
    • 设置完成后,找到“Exit”或“Save & Exit”选项,选择“Save Changes and Exit”保存设置并退出。
  3. 选择“Yes”确认保存BIOS设置。电脑将会重新启动。
  4. 开机时将根据新设置的启动优先级从高优先设备启动。

需要注意的是,不同主板的BIOS菜单选项可能有所差异,需要找到与启动相关的设置项目进行修改。保存后要重新启动电脑使设置生效。

插入U盘，启动电脑，进行安装

按照提示操作即可，可能要删除磁盘来分配为系统盘。

关于GPT/MBR的区分

GPT和MBR是两种不同的分区方案,区分它们的方法如下:

  1. 使用 diskpart 命令

在命令提示符下输入diskpart,然后输入list disk和list volume命令,如果显示表明磁盘分区方案是GPT,则为GPT分区。如果没有显示,则为MBR分区。

  1. 使用 diskmgmt.msc

右键点击内置磁盘管理工具底部的磁盘条目,选择“属性”,如果分区样式显示为GPT则为GPT分区,反之则是MBR。

  1. 使用第三方磁盘管理工具

在一些第三方磁盘分区管理工具中,也可以直接显示磁盘分区模式。如DiskGenius、AOMEI Partition Assistant等。

  1. 通过分区数量判断

GPT最多支持128个主分区,而MBR最多支持4个主分区。如果一个磁盘上分区数量大于4个,则很有可能是GPT分区。

  1. 通过磁盘大小判断

GPT对磁盘大小无限制,但MBR的最大支持容量为2TB。因此,如果磁盘容量大于2TB,其分区模式一般是GPT。

综合以上几点判断,可以比较准确地区分出GPT和MBR两种分区模式。
