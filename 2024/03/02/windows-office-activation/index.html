Windows / Office 激活教程

发表于2024-03-02|更新于2024-05-12|Windows-Software
|阅读量:

我们都知道 Windows 与 Office 的买断制价格通常是算在购买电脑的费用中的, 然而每当我们重装系统后, 往往需要重新激活 Office 与 Windows, 本文将介绍如何通过 MAS 激活 Windows 与 Office.

本文主要介绍的工具是 Microsoft Activation Scripts(MAS), 其 Github 仓库地址是 massgravel/Microsoft-Activation-Scripts.

安装 Windows

请参考另一篇blog的这个章节.

安装 Office

推荐使用 Office Tool Plus.

  1. 访问此工具的下载页面, 下载对应的版本并解压运行.
  2. 在工具内选择需要安装的 Office 版本, 然后进行部署.

激活 Windows 与 Office

注意: 激活 Office 前, 最好确保你没有登录微软账户, 如果有登录, 最好先注销账户.
